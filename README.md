# DayHeatmap

一日の充実度を1〜5で評価して、GitHub芝のようなヒートマップで可視化するiOSウィジェット。

[Scriptable](https://scriptable.app/) (無料) で動作します。

## スクリーンショット

（後で追加予定）

## 機能

- **月ビュー**: 当月のカレンダー形式でヒートマップ表示
- **年ビュー**: 12ヶ月分を一覧表示
- ウィジェット更新ごとに月/年ビューが交互に切り替わる
- 連続記録日数（ストリーク）表示
- 今日のセルをハイライト

## セットアップ

### 1. Scriptableをインストール
App Storeで「Scriptable」を検索してインストール（無料）

### 2. スクリプトを追加
1. `DayHeatmap.js` の中身をコピー
2. Scriptableアプリを開く → 右上「+」→ 新規スクリプト
3. 名前を `DayHeatmap` に変更
4. コードを貼り付けて保存

### 3. ウィジェットを配置
1. ホーム画面を長押し → 左上「+」
2. Scriptable を検索 → **Medium** サイズを選択
3. ウィジェットをタップ → Script に「DayHeatmap」を選択

### 4. 毎日の記録
Scriptableアプリで DayHeatmap をタップ → 1〜5で評価

#### リマインド設定（おすすめ）
iOSショートカットアプリで毎晩の通知を設定:
1. ショートカット → オートメーション → 「時刻」→ 毎日22:00
2. アクション: 「Appを開く」→ Scriptable

## 評価スケール

| スコア | 色 | 意味 |
|--------|-----|------|
| 未記録 | 暗いグレー | — |
| 1 | 暗い緑 | ダメだった |
| 2 | 緑 | いまいち |
| 3 | 明るい緑 | ふつう |
| 4 | 鮮やかな緑 | よかった |
| 5 | 金色 | 最高！ |

## データ

評価データは Scriptable の iCloud ドキュメントフォルダに `day_heatmap.json` として保存されます。

```json
{
  "2026-04-08": 4,
  "2026-04-07": 3
}
```
